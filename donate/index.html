-- About --

The Code Cooperative is an open source educational program that teaches coding as a means to creating social change. We are partnered with The Fortune Society, a 501(c)(3) non profit organization.

-- Donate --

Access to a computer and the Internet is crucial to learning to code, which is why we are seeking laptop donations from tech companies. Laptops would be donated to the Fortune Society (100% tax-deductible).
